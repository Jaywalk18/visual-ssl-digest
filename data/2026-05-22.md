## 阅读路线

先读 **Multimodal LLMs under Pairwise Modalities**，它最接近通用视觉-语言/多模态自监督预训练问题，尤其是“pair-wise contrastive + self-modal reconstruction”的潜空间设计。然后读 **Weighted Reverse Convolution**，它是 replacement 更新，但对 DINO/CLIP/SigLIP 等 VFM 的 dense feature 可用性很直接。第三读 **AIR**，它偏重自监督图像重建和 2D Gaussian Splatting，和通用表征学习的联系偏间接但可启发重建式预训练。**Capability != Interpretability** 建议作为诊断论文扫读，用来反思 DINO/CLIP/SigLIP 表征质量评估。FullFlow、Latent Actions、SGA 更偏多模态生成、视频世界模型或生成先验蒸馏，快速扫方法即可。

- 核心方法：把多模态统一建模拆成可扩展的 pairwise modality 学习。第一阶段通过自模态重建与模态对之间的对比学习学习共享潜空间；第二阶段把新模态 encoder 接入已预训练模态 decoder，实现跨模态迁移与生成。
- 主要贡献：试图避免每加入一个新模态就重训完整 MLLM，用 pairwise 对齐和最小潜变量约束扩展到 3D 点云、触觉等新增模态。
- 实验亮点：在三个模态对上验证新增 3D 点云和触觉模态后的跨模态性能，强调 latent representation space 的复用。
- 局限性：摘要层面还看不到对纯视觉表征 benchmark 的充分验证；新增模态覆盖偏机器人/触觉，和通用图像 SSL 仍有距离。
- 为什么相关：这是今日最贴近“视觉-语言/多模态自监督预训练”的条目；自模态重建 + pairwise contrastive learning 的组合对通用视觉表征学习有直接参考价值。

### P2 - AIR: Self-supervised Feed-forward Reconstruction of Complex Images with 2D Gaussian Splatting

- 标题：[AIR: Self-supervised Feed-forward Reconstruction of Complex Images with 2D Gaussian Splatting](https://arxiv.org/abs/2605.20820)
- 作者/机构：Xianda Guo, Jinheng Xie, Zhaoyang Lv, Zhenbei Liu, Hui Zhang, Yujun Shen, et al.
- 来源链接：[arXiv:2605.20820](https://arxiv.org/abs/2605.20820)
- 发布日期或会议信息：arXiv cs.CV new listing，2026-05-21。
- 核心方法：提出 Alignment-Invariant Representation，把复杂图像的前馈重建和 2D Gaussian Splatting 表示结合，目标是摆脱强监督或逐图优化。
- 主要贡献：把 2DGS 用作图像重建的结构化中间表示，尝试用自监督方式恢复复杂图像内容。
- 实验亮点：摘要称方法在复杂图像前馈重建上提升重建质量与效率，适合关注重建式视觉表征的读者进一步查实验细节。
- 局限性：更像“图像重建/表示压缩”而不是典型 ImageNet-style 表征预训练；下游分类、检测、分割迁移效果需要重点核实。
- 为什么相关：重建式预训练仍是 MIM/MAE 系列的重要思想来源；AIR 的结构化重建目标可能启发更细粒度的视觉自监督 pretext。

### P2 - Capability != Interpretability: Human Interpretability of Vision Foundation Models

- 标题：[Capability != Interpretability: Human Interpretability of Vision Foundation Models](https://arxiv.org/abs/2605.20337)
- 作者/机构：Julien Colin, Lore Goetschalckx, Nuria Oliver, Thomas Serre
- 来源链接：[arXiv:2605.20337](https://arxiv.org/abs/2605.20337)
- 发布日期或会议信息：arXiv cs.CV new listing，2026-05-21。
- 核心方法：用 sparse autoencoder 恢复特征，再通过 localizability 与 nameability 两个心理物理协议评估视觉模型特征是否可被人类解释。
- 主要贡献：把 DINOv2、DINOv3、CLIP、SigLIP 与监督 ViT 放在同一可解释性标尺上，提出 capability 与 interpretability 不应混为一谈。
- 实验亮点：覆盖 6 个 ViT、超过 15,000 条行为响应；摘要称 foundation models 的人类可解释性弱于监督基线，且可解释性与下游性能不相关。
- 局限性：这是诊断/评估论文，不提出新的自监督训练目标；人的命名和定位协议成本高，是否能规模化为日常 VFM 评测还需观察。
- 为什么相关：对通用视觉 SSL 而言，线性 probe/迁移性能不足以说明表征质量；该文给出一个评估 DINO/CLIP/SigLIP 特征语义结构的新视角。

### P3 - SGA: Distilling Sam-DINO Features in High-Resolution Image Generation via 2D Gaussian Splatting

- 标题：[SGA: Distilling Sam-DINO Features in High-Resolution Image Generation via 2D Gaussian Splatting](https://arxiv.org/abs/2605.20808)
- 作者/机构：Zhen Liu, Ruijie Quan, Han Zhou, Zhiguo Cao, et al.
- 来源链接：[arXiv:2605.20808](https://arxiv.org/abs/2605.20808)
- 发布日期或会议信息：arXiv cs.CV new listing，2026-05-21。
- 核心方法：把 SAM/DINO 的结构性视觉先验蒸馏到高分辨率图像生成中，通过 2D Gaussian Splatting 与 spatial Gram alignment 保持空间结构。
- 主要贡献：给生成模型引入来自自监督/基础视觉模型的 dense semantic prior，缓解高分辨率生成中的结构漂移。
- 实验亮点：摘要强调在高分辨率生成中提升结构一致性和局部细节，适合关注 VFM 特征作为 teacher 的读者扫读。
- 局限性：主目标是图像生成而不是视觉 encoder 预训练；对通用表征的反向收益没有直接证明。
- 为什么相关：DINO/SAM 作为 teacher 的使用方式值得关注，尤其是如何把 dense visual priors 注入新模型。

### P3 - FullFlow: Upgrading Text-to-Image Flow Matching Models for Bidirectional Vision-Language Generation

- 标题：[FullFlow: Upgrading Text-to-Image Flow Matching Models for Bidirectional Vision--Language Generation](https://arxiv.org/abs/2605.20316)
- 作者/机构：Eric Tillmann Bill, Enis Simsar, Alessio Tonioni, Thomas Hofmann
- 来源链接：[arXiv:2605.20316](https://arxiv.org/abs/2605.20316)
- 发布日期或会议信息：arXiv cs.CV new listing，2026-05-21；项目页见 arXiv 备注。
- 核心方法：在预训练 rectified-flow T2I 模型上只训练 LoRA adapter 和轻量 text heads，引入离散文本插入过程，实现 text-to-image、image-to-text、joint sampling 和 partial-text prediction。
- 主要贡献：证明不必完整多模态预训练，也能从现有 T2I flow 模型中解锁双向视觉-语言能力。
- 实验亮点：摘要报告 SD3 上 FID 由 62.7 降至 31.6，image-to-text CIDEr 由 2.0 到 99.4，显存从约 84GB 降至约 38GB，并迁移到 FLUX.1-dev。
- 局限性：偏生成模型适配；视觉 encoder 的表征质量、下游视觉迁移和自监督 pretext 不是核心评估。
- 为什么相关：对多模态预训练路线有启发：强视觉生成先验也许能通过轻量训练转化为视觉-语言表示能力。

### P3 - Why Latent Actions Fail, and How to Prevent It

- 标题：[Why Latent Actions Fail, and How to Prevent It](https://arxiv.org/abs/2605.20223)
- 作者/机构：Jung Min Lee, Taehyun Cho, Li Zhao, Jungwoo Lee
- 来源链接：[arXiv:2605.20223](https://arxiv.org/abs/2605.20223)
- 发布日期或会议信息：arXiv cs.CV new listing，2026-05-21。
- 核心方法：分析 latent action models 从无标注视频中压缩 frame-to-frame changes 时为什么会把背景等 exogenous state 编进动作表示，并给出表示空间中聚焦 endogenous components 的理论解释。
- 主要贡献：用线性 LAM 框架解释标准重建目标的失败模式，以及动作监督等 auxiliary objective 为什么有帮助。
- 实验亮点：摘要称在线性与非线性 LAM 上验证理论发现，为视频自监督/世界模型中的 latent action 学习提供诊断框架。
- 局限性：偏 embodied/world-model latent action，而不是通用图像 encoder；是否迁移到 ImageNet/视频理解预训练仍未知。
- 为什么相关：通用视频自监督常遇到背景捷径和动态因素混淆，该文对“哪些变化该进表示”给出可借鉴分析。

### P3 - Can Vision Models Truly Forget? Mirage: Representation-Level Certification of Visual Unlearning

- 标题：[Can Vision Models Truly Forget? Mirage: Representation-Level Certification of Visual Unlearning](https://arxiv.org/abs/2605.20282)
- 作者/机构：Zhenyu Yu, Yangchen Zeng, Chunlei Meng, Guangzhen Yao, Shuigeng Zhou
- 来源链接：[arXiv:2605.20282](https://arxiv.org/abs/2605.20282)
- 发布日期或会议信息：arXiv cs.CV new listing，2026-05-21。
- 核心方法：提出 Mirage，用 Linear Probe Recovery、CKA、Feature Separability Scoring、Layer-wise Recovery Analysis 审计视觉模型是否在 representation level 真正遗忘。
- 主要贡献：指出只看输出层 unlearning 会漏掉表征空间中残留的类别结构。
- 实验亮点：覆盖 7 个数据集和 7 个 baseline；摘要称 class-level forgetting 仍可能保留强表征痕迹，而 sample-level forgetting 接近随机。
- 局限性：主题是 federated unlearning，不是自监督训练方法；但工具可用于审计 SSL/VFM 表征残留。
- 为什么相关：通用视觉表征越强，越需要表征层面的隐私、遗忘和安全评估；该文可作为 VFM representation audit 工具参考。

### 更新/状态变化 - Weighted Reverse Convolution for Feature Upsampling

- 标题：[Weighted Reverse Convolution for Feature Upsampling](https://arxiv.org/abs/2605.17472)
- 作者/机构：Wentong Li, Zhiyuan Qi, Zichen Zhao, Kai Zhang, Lei Zhang
- 来源链接：[arXiv:2605.17472](https://arxiv.org/abs/2605.17472)
- 发布日期或会议信息：arXiv replacement，2026-05-21 列入 replacements；未按新增重复计数。
- 核心方法：把 VFM feature upsampling 表述为带空间可变权重的 Tikhonov 正则最小二乘逆问题，并给出可微闭式 FFT 解。
- 主要贡献：作为 drop-in upsampling operator，把粗 patch-level VFM 特征密化，服务 dense prediction、point-wise correspondence 等任务。
- 实验亮点：摘要称集成到轻量 self-supervised densification framework 后，在 segmentation、depth estimation、video object segmentation、object discovery、keypoint correspondence 上稳定提升。
- 局限性：需要看正文确认对不同 VFM、不同分辨率和极端小目标的稳定性；replacement 更新未必代表新方法变化。
- 为什么相关：这是今日最值得跟踪的 VFM dense representation 工具型更新，适合接在 DINO/CLIP/SigLIP 等自监督或弱监督视觉 backbone 后。

## 趋势观察

1. **从“训练更大 encoder”转向“修补和审计 VFM 表征”**：WRC、Mirage、Capability != Interpretability 都在处理已有 VFM 表征的密化、遗忘、可解释性，而不是重新提出 SSL backbone。
2. **多模态自监督仍在找可扩展接口**：Pairwise Modalities 和 FullFlow 都试图避免完整重训，用共享潜空间或轻量 adapter 把新能力接到已有模型上。
3. **DINO/SAM/CLIP/SigLIP 正在成为默认 teacher 或审计对象**：今天多篇论文不是训练它们，而是蒸馏、测量或利用它们的 dense/semantic priors。
4. **垂直域 foundation model 继续增多**：遥感、医学、工业检测、驾驶和机器人条目很多，但大多数不应进入通用视觉自监督精读队列。
