## 阅读路线

先读 TC-JEPA，它直接站在 I-JEPA/Masked feature prediction 主线上，核心问题是“遮挡位置预测不确定性如何靠文本条件缓解”。第二读 ResDreamer，关注层级残差世界模型如何在纯自监督条件下学习动态视觉推理 latent。然后读 Cambrian-P 和 Directional Motion Blindness：前者把相机位姿变成视频 MLLM 的基础监督信号，后者诊断运动方向信号在视觉编码器到语言读出之间如何断裂。SWEET、GeoWeaver 和 Matching Principle 更适合作为方法库：分别对应可缩放初始化、几何 token grounding 和鲁棒表征损失理论。

## 会议/顶会动态

### Text-Conditional JEPA for Learning Semantically Rich Visual Representations

- **作者/机构**：Chen Huang、Xianhang Li、Vimal Thilak、Etai Littwin、Josh Susskind；Apple Machine Learning Research。
- **来源链接**：[arXiv:2605.03245](https://arxiv.org/abs/2605.03245)，[Apple Research 页面](https://machinelearning.apple.com/research/text-conditional-jepa-visual-representations)。
- **发布日期或会议信息**：arXiv submitted 2026-05-05；arXiv comments 与 Apple 页面均标注 ICML 2026。
- **核心方法**：在 I-JEPA 的 masked feature prediction 上加入 caption 条件，用 fine-grained text conditioner 对输入文本 token 做稀疏 cross-attention，从而调制待预测 patch features。
- **主要贡献**：把 JEPA 从纯视觉 masked latent prediction 扩展为“feature prediction only”的视觉语言预训练范式，避免完全依赖对比学习。
- **实验亮点**：摘要称提升下游表现与训练稳定性，并在细粒度视觉理解/推理任务上超过对比方法。
- **局限性**：需要 caption 条件，严格意义上不再是纯视觉 SSL；文本质量和跨模态对齐会影响学习目标。
- **为什么相关**：这是今天最值得精读的通用视觉自监督主线工作，直接连接 I-JEPA、masked prediction 和视觉语言预训练。

### Self-Supervised Weight Templates for Scalable Vision Model Initialization

- **作者/机构**：Yucheng Xie、Fu Feng、Ruixiao Shi、Jing Wang、Yong Rui、Xin Geng。
- **来源链接**：[arXiv:2601.19694](https://arxiv.org/abs/2601.19694)。
- **发布日期或会议信息**：arXiv submitted 2026-01-27；去重库补录 ICML 2026 Poster 证据。
- **核心方法**：SWEET 学习共享 weight template 与 size-specific weight scalers，基于 Tucker factorization 支持不同深度/宽度架构初始化；用 width-wise stochastic scaling 增强跨宽度泛化。
- **主要贡献**：把自监督预训练从“训练一个固定模型”改成“学习可组合的初始化模板”，服务多尺寸视觉模型部署。
- **实验亮点**：摘要称在 classification、detection、segmentation、generation 上为可变尺寸视觉模型初始化达到 SOTA。
- **局限性**：与表示学习的关系在权重空间而非特征空间；需要看全文确认自监督约束和不同架构族覆盖范围。
- **为什么相关**：如果关注视觉基础模型的可扩展预训练和模型族蒸馏/初始化，这是很有方法价值的支线。

### Self-supervised Hierarchical Visual Reasoning with World Model

- **作者/机构**：Yuanfei Xu、Lin Liu、Wengang Zhou、Mingxiao Feng、Houqiang Li。
- **来源链接**：[arXiv:2605.17537](https://arxiv.org/abs/2605.17537)。
- **发布日期或会议信息**：arXiv 2026-05；去重库补录 ICML 2026 Poster 证据。
- **核心方法**：ResDreamer 使用层级世界模型，高层重构低层残差，逐层抽象世界动态；高层残差表征反过来调制低层预测。
- **主要贡献**：在纯自监督条件下学习 visual foresight reasoning representations，强调任务相关信号比 photorealistic fidelity 更重要。
- **实验亮点**：摘要称在 3D open-world RL 场景中实现 SOTA sample efficiency 和 parameter efficiency。
- **局限性**：主要在 RL/3D open-world 环境验证，和通用图像 benchmark 的直接对应有限。
- **为什么相关**：它是“视频/世界模型式 SSL”路线的重要样本，可借鉴层级残差目标设计。

### From Abstraction to Instantiation: Learning Behavioral Representation for Vision-Language-Action Model

- **作者/机构**：Bing Hu、Zaijing Li、Rui Shao、Junda Chen、April Hua Liu、Wei-Shi Zheng、Liqiang Nie。
- **来源链接**：[arXiv:2605.22671](https://arxiv.org/abs/2605.22671)。
- **发布日期或会议信息**：arXiv submitted 2026-05-21；arXiv 页面 Journal reference 标注 ICML 2026 Spotlight。
- **核心方法**：BehaviorVLA 通过 Visuomotor Behavior Encoder 聚合长程轨迹为统一行为表征，再用 Phase-conditioned Behavior Decoder 按执行阶段解码动作。
- **主要贡献**：把 VLA 泛化问题转成时间一致行为表征学习，而不是只建模短视 action latent。
- **实验亮点**：摘要报告 RoboTwin 2.0、LIBERO、CALVIN 上达到 SOTA；真实 sim-to-real 中用 50% demonstration data 匹配 OpenVLA-OFT。
- **局限性**：机器人操作场景强，视觉自监督成分间接。
- **为什么相关**：可借鉴其长程视觉-动作 latent 聚合方式，但不应排在核心图像 SSL 前列。

### DINOv3

- **作者/机构**：Oriane Siméoni、Huy V. Vo、Maximilian Seitzer、Maxime Oquab、Andrea Vedaldi 等；Meta/合作机构。
- **来源链接**：[OpenReview TMLR 页面](https://openreview.net/forum?id=2NlGyqNjns)，[代码](https://github.com/facebookresearch/dinov3)，[Meta 博客](https://ai.meta.com/blog/dinov3-self-supervised-vision-model/)。
- **发布日期或会议信息**：OpenReview 标注 Published / Last Modified: 2026-05-15，Accepted by TMLR。
- **核心方法**：扩展 DINO 系列自监督训练，引入 Gram anchoring 缓解长训中 dense feature map 退化，并用后处理策略提升分辨率、模型尺寸和文本对齐灵活性。
- **主要贡献**：给出一套通用视觉 foundation model，目标是在多源数据上学习高质量 dense features。
- **实验亮点**：OpenReview 摘要称在多类视觉任务上超过此前自监督/弱监督 foundation models，且无需任务级微调。
- **局限性**：发布时间超过本次严格 48 小时窗口；今日只做状态补录。
- **为什么相关**：高度相关，是后续 dense prediction、VLM grounding、RAE 论文的重要基础特征源。

## arXiv 新增/更新

### Cambrian-P: Pose-Grounded Video Understanding

- **作者/机构**：Jihan Yang、Zifan Zhao、Xichen Pan、Shusheng Yang、Junyi Zhang、Bingyi Kang、Hu Xu、Saining Xie。
- **来源链接**：[arXiv:2605.22819](https://arxiv.org/abs/2605.22819)。
- **发布日期或会议信息**：Submitted 2026-05-21；出现在 arXiv cs.CV 2026-05-22 new listing。
- **核心方法**：在视频 MLLM 中加入 per-frame learnable camera tokens 和 pose regression head，用相机位姿建立跨帧共享空间坐标。
- **主要贡献**：把位姿从下游几何任务信号提升为视频理解的通用监督/自监督式约束。
- **实验亮点**：摘要报告 VSI-Bench 等空间推理 benchmark 提升 4.5-6.5%，并泛化到 8 个额外空间和通用视频 QA benchmark；ScanNet streaming pose estimation 达到 SOTA。
- **局限性**：依赖伪位姿质量和采样策略；对纯图像表征学习的直接迁移仍需验证。
- **为什么相关**：位姿/几何 pseudo-label 可成为视频 SSL 或视觉基础模型预训练中的廉价跨帧结构信号。

### Which Way Did It Move? Diagnosing and Overcoming Directional Motion Blindness in Video-LLMs

- **作者/机构**：Jongseo Lee、Hyuntak Lee、Sunghun Kim、Sooa Kim、Jihoon Chung、Jinwoo Choi。
- **来源链接**：[arXiv:2605.22823](https://arxiv.org/abs/2605.22823)。
- **发布日期或会议信息**：Submitted 2026-05-21；Preprint。
- **核心方法**：追踪运动方向信息在 vision encoder、projector、LLM hidden states 中的线性可读性；提出 MoDirect 数据族和 DeltaDirect projector-level objective，从相邻帧 feature delta 预测归一化 2D motion vectors。
- **主要贡献**：证明很多 Video-LLM 的问题不是完全“看不到运动”，而是读出端无法把运动方向绑定到正确语言答案。
- **实验亮点**：摘要报告 MoDirect-SynBench 中 DeltaDirect 将运动方向准确率从 25.9% 提升到 85.4%；MoDirect-RealBench 相比 vanilla baseline 提升 21.9 个百分点。
- **局限性**：任务聚焦 signed image-plane motion direction，距离完整视频语义预训练仍有差距。
- **为什么相关**：它给出一个适合视觉自监督研究的问题切片：视觉特征中已有的信息如何在多模态接口处丢失或错绑。

### GeoWeaver: Grounding Visual Tokens with Geometric Evidence before Scene Reasoning

- **作者/机构**：Deshui Miao、Xingsen Huang、Yameng Gu、Xin Li、Haijun Zhang、Ming-Hsuan Yang。
- **来源链接**：[arXiv:2605.22558](https://arxiv.org/abs/2605.22558)。
- **发布日期或会议信息**：Submitted 2026-05-21。
- **核心方法**：从冻结几何编码器构造 multi-level geometry bank，并按 token 自适应分配几何证据，通过 residual grounding 在进入语言模型前改造视觉 token。
- **主要贡献**：强调几何不是 reasoning-stage 后融合辅助分支，而是视觉 token 的表示前提。
- **实验亮点**：摘要称在多个 spatial reasoning benchmarks 上持续提升 geometry-aware reasoning，同时保持一般多模态能力。
- **局限性**：依赖冻结 geometry encoder 的覆盖面；开放域几何估计不稳时可能放大误差。
- **为什么相关**：代表“语义特征 + 几何基础”的预训练/适配方向，适合视频和 3D-aware VLM。

### Swift Sampling: Selecting Temporal Surprises via Taylor Series

- **作者/机构**：Dahye Kim、Bhuvan Sachdeva、Karan Uppal、Naman Gupta、Vineeth N. Balasubramanian、Deepti Ghadiyaram。
- **来源链接**：[arXiv:2605.22678](https://arxiv.org/abs/2605.22678)。
- **发布日期或会议信息**：Submitted 2026-05-21。
- **核心方法**：把长视频视作视觉 latent space 中的可微轨迹，计算速度/加速度并用 Taylor expansion 预测后续路径，选择偏离预测流形最大的 temporal surprise frames。
- **主要贡献**：给出无需训练、无需额外网络、低开销的视频帧选择策略。
- **实验亮点**：摘要称额外计算仅约 baseline 的 0.02x，有限帧预算下最高提升 12.5 points。
- **局限性**：不直接学习新视觉表征；高度依赖底层视觉 latent 的质量。
- **为什么相关**：可作为视频 SSL 数据采样、长视频评测和多模态预训练 curriculum 的轻量模块。

### The Matching Principle: A Geometric Theory of Loss Functions for Nuisance-Robust Representation Learning

- **作者/机构**：Vishal Rajput。
- **来源链接**：[arXiv:2605.22800](https://arxiv.org/abs/2605.22800)。
- **发布日期或会议信息**：Submitted 2026-05-21。
- **核心方法**：把 robustness、domain adaptation、augmentation、metric learning、Jacobian penalties 等统一为估计 label-preserving deployment nuisance covariance，并沿覆盖该协方差的矩阵范围正则化 encoder Jacobian。
- **主要贡献**：提出 matching principle 和 Trajectory Deviation Index，试图给 nuisance-robust representation learning 的损失函数选择一个可证伪几何框架。
- **实验亮点**：摘要称 13 个预注册实验块中 12 个符合 matched > isotropic > wrong-W 的预测排序，并扩展到 7B 模型风格对齐实验。
- **局限性**：理论/经验跨度很大，视觉 SSL 直接可用性需要谨慎验证；单作者长文应等待社区复核。
- **为什么相关**：对自监督表征的增广不变性、域泛化和鲁棒损失设计有潜在理论价值。

## 趋势观察

1. JEPA 路线正在和视觉语言条件结合：TC-JEPA 说明 masked feature prediction 可能不必走纯视觉或对比学习二选一。
2. 视频/世界模型表征开始强调物理结构信号，包括位姿、运动方向、几何 token evidence 和 temporal surprise。
3. 自监督对象不再只限于 activation representation：SWEET 把预训练目标推进到权重模板和可变尺寸模型初始化。
4. DINOv3 的 TMLR 接收会继续影响 dense prediction、VLM grounding、RAE 和生成模型 latent encoder 选择。
