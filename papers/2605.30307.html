核心问题

视觉基础模型需要更强的 region-level 与 3D-aware grounding，GR3D 提供了可参考的系统设计。

方法拆解

在同一 VLM 中集成 explicit 2D grounding、implicit 2D grounding 与 monocular 3D grounding；生成时插入 region tokens 支持 spatial chain-of-thought。

主要贡献

把 grounded 2D perception 与 3D inference 组合成可解释空间推理流程。

实验看点

摘要称 grounded 与 non-grounded spatial benchmark 都有提升；重点看是否真的来自 3D grounding 而非额外区域监督。

局限与读法

监督和任务设计较重，主要是 VLM reasoning 系统；和纯自监督视觉表征有距离。
