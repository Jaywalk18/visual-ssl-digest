核心问题

它说明 object-centric visual evidence 如何在 VLM 中被路由和复用，对未来通用视觉表征的“可查询性”有参考价值。

方法拆解

每次 object grounding 预测后，ROVER 注入 step-specific token triplet，把当前 reasoning context、object-centric differential attention 和历史跨图证据路由进 visual working space。

主要贡献

避免单纯裁剪 RoI 或注入 RoI features 导致全局场景和对象关系被削弱，同时降低多图、多对象推理上下文成本。

实验看点

摘要称集成到 Qwen2.5-VL-7B 后，MM-GCoT answer accuracy +4.8%、grounding accuracy +14.6%，VideoEspresso answer accuracy +8.6%。

局限与读法

依赖 grounding prediction 和 SFT-to-GRPO pipeline；它优化的是推理流程，不是底层视觉 encoder 的无监督学习。
