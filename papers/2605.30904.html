核心问题

视觉 tokenizer 正成为图像/视频 foundation model 底层接口，语义化 token 对 MIM、AR、diffusion 都重要。

方法拆解

在 encoder-decoder 中联合优化 VAE/VQ tokenizer，通过相似 token clustering 提供 semantic alignment 与 group-wise 约束。

主要贡献

尝试同时获得 VAE 重建质量和 VQ 生成友好离散性，并让 latent 更语义化。

实验看点

摘要报告 ImageNet-256 上重建和生成有竞争力，匹配 token budget 下 rFID 低于强 VAE/VQ baselines。
