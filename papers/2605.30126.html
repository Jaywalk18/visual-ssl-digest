先说结论。视觉 token 是 VLM 中通用视觉表征的接口，压缩策略会直接决定哪些视觉信息进入语言模型。 指出纯空间压缩会 aliasing，纯 query 压缩会损失 spatial grounding；提出两类 token 分工来缓解表示冲突。

核心问题

视觉 token 是 VLM 中通用视觉表征的接口，压缩策略会直接决定哪些视觉信息进入语言模型。

方法拆解

建立 Pool-Anchored Resampling：spatial pool tokens 保留低频布局，conditioned elastic queries 汇聚细粒度非局部信息。

主要贡献

指出纯空间压缩会 aliasing，纯 query 压缩会损失 spatial grounding；提出两类 token 分工来缓解表示冲突。

实验看点

摘要强调 aggressive compression 下更稳；重点看 VQA/OCR/grounding 的保真曲线与 token budget sweep。

局限与读法

效率架构多于新预训练目标；训练成本、不同视觉 encoder 兼容性和视频场景仍需验证。
