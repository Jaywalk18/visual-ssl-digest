核心问题

虽然不是无标签 SSL，但它直指视觉 token 表征学习瓶颈：VLM 的“看见”不应只靠文本答案反向传播。

方法拆解

利用 OCR 场景中的视觉 patch 与文字对应关系，自动给每个 visual token 标注对应 word，并仍用 next-token prediction 训练，不加 decoder 或额外 forward。

主要贡献

把 fine-grained visual grounding 从“答案文本监督”推进到“视觉 token 级监督”，给 VLM 后训练提供了一种更直接的视觉信号。

实验看点

摘要称 DV-SFT 在 3 个域内和 4 个域外 benchmark 上稳定优于标准 SFT，并提升多模态 alignment efficiency。

局限与读法

监督构造依赖 OCR/文字可见场景，能否扩展到自然图像细粒度物体、关系和几何 token 仍是关键问题。
