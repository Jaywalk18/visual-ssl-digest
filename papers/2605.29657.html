核心问题

作为 visual token 选择机制，它能帮助判断哪些 token 在预训练 VLM 表征中真正携带信息。

方法拆解

不用绝对 top-K 重要性排序，而以 register tokens 为低信息参考，判断每个 visual token 是否提供额外证据，并自适应预算。

主要贡献

指出 attention sinks 和固定预算会让绝对排序脆弱；以相对证据测试替代。

实验看点

摘要主打 prefill 计算和 memory 降低；重点看 open-ended VQA、OCR 与小目标场景是否稳定。

局限与读法

训练-free 剪枝不是新表征学习；偏推理系统。
