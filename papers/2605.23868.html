先说结论。直接触及自监督 ViT 的 token-level dense transfer 问题，对分割、检测和开放词汇 dense prediction 都有启发。 指出 dense representation 质量不能只用 locality 解释，CLS/global context 与 patch token 的选择性交互同样重要。

核心问题

直接触及自监督 ViT 的 token-level dense transfer 问题，对分割、检测和开放词汇 dense prediction 都有启发。

方法拆解

重新分析 ViT prolonged training 后 patch representation 的 dense degradation，提出“semantic diffusion”解释，并用 entmax-1.5 sparse attention 替代 softmax attention。

主要贡献

指出 dense representation 质量不能只用 locality 解释，CLS/global context 与 patch token 的选择性交互同样重要。

实验看点

在 DINOv1 ViT-S/16 ImageNet-1K 200 epoch 设置下，保持 ImageNet linear probing，同时 VOC mIoU 42.80 -> 48.78，ADE20K 19.85 -> 21.97，Cityscapes 36.79 -> 37.87。

局限与读法

实验规模偏小，主要是 DINOv1 ViT-S/16；是否能推广到 DINOv2/DINOv3、MAE、SigLIP 等大模型还需验证。
